# SpiceIsleTours

Student final project: A small tour booking website built with PHP, MySQL, HTML, CSS and JS.

## Setup
1. Create a MySQL database and user.
2. Edit `includes/db.php` with your DB credentials.
3. Run `sql/init.sql` in your database client to create tables and sample data.
4. Upload files to a PHP-enabled host (or run locally with XAMPP / MAMP / LocalWP).
